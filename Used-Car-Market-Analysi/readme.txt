# Used Car Market Analysis Dashboard

## Project Overview

An interactive Tableau dashboard analyzing a dataset of 1M+ used car records.

## Features

- Average Car Price Trend (2000–2024)
- Total Price by Brand and Model
- Average Price by Brand and Color
- Interactive filters
- KPI cards

## Tools Used

- Tableau Public
- CSV Dataset
- Data Visualization
- Dashboard Design

## Insights

- Luxury brands exhibit higher average prices.
- Price trends increase steadily over time.
- Vehicle color shows variations in average resale prices.
